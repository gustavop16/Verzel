# VERZEL

* Teste técnico | PHP + LARAVEL + REACT/ANGULAR

# Tecnologia

* PHP 8.1
* Laravel Framework 10
* MySql

# Pré-requisitos

Certifique-se de ter as seguintes ferramentas instaladas em seu computador:

- [PHP](https://www.php.net/) (>= 8.1)
- [Composer](https://getcomposer.org/)
- [MySQL](https://www.mysql.com/) ou qualquer outro banco de dados que você planeja usar

# Instalação

* $ composer install
* $ cp .env.example .env

- configure as variáveis de ambiente, especialmente as configurações do banco de dados:

* $ php artisan key:generate
* $ php artisan migrate
* $ php artisan serve
